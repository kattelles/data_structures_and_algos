## Static Array

Very basic implementation of a static array.

| API           | BigO (worst case)
| --------------|:---------------:|
| [](idx)       | O(1)            |
| []=(idx, val) | O(1)            |


## Dynamic Array

Built upon a static array, a Dynamic Array will resize! when necessary.

| API           | BigO (worst case)
| --------------|:---------------:|
| [](idx)       | O(1)            |
| []=(idx, val) | O(1)            |
| pop           | O(1)            |
| push(val)     | O(n)            |
| shift         | O(n)            |
| unshift(val)  | O(n)            |
| resize!       | O(n)            |

## Ring Buffer

The Ring Buffer is an optimization of the Dynamic Array.

| API           | BigO (worst case)
| --------------|:---------------:|
| [](idx)       | O(1)            |
| []=(idx, val) | O(1)            |
| pop           | O(1)            |
| push(val)     | O(1)            |
| shift         | O(1)            |
| unshift(val)  | O(1)            |
| resize!       | O(n)            |
